# Bug Explainer

![Status](https://img.shields.io/badge/status-live-f59e0b) ![Gemini](https://img.shields.io/badge/AI-Gemini%202.0%20Flash-4285F4) ![HTML](https://img.shields.io/badge/built%20with-HTML%2FJS-orange)

> Agent #8 of 97 | 100 AI Agents Challenge

Paste your buggy code + error message and get an **animated step-through execution trace** — watch your code run line by line, see variable state at every step, and identify the exact crash point. Plus a plain-English explanation tailored to your experience level.

## Features
- **Step-Through Execution Trace** — animate code line by line with play/pause/step controls
- **Live variable state panel** — see what every variable holds at each step
- **Crash highlighting** — the exact error line pulses red with a plain-English note
- Diff-style view showing old vs fixed code
- Tailored explanations (beginner → advanced) with real-world analogies
- Follow-up chat for deeper understanding
- Bug history (last 5 bugs saved locally)
- Supports Python, JavaScript, Java, C++, TypeScript
