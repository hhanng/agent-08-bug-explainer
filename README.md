# Bug Explainer

![Status](https://img.shields.io/badge/status-live-f59e0b) ![Gemini](https://img.shields.io/badge/AI-Gemini%202.0%20Flash-4285F4) ![HTML](https://img.shields.io/badge/built%20with-HTML%2FJS-orange)

> Agent #8 of 97 | 100 AI Agents Challenge

Paste your buggy code + error message and get a plain-English explanation of exactly what went wrong, why it happened, and the exact fix — tailored to your experience level.

## Features
- Explains bugs with real-world analogies for beginners
- Diff-style view showing old vs fixed code
- Tailored explanations for beginner → advanced levels
- Follow-up chat for deeper understanding
- Supports Python, JavaScript, Java, C++, TypeScript
